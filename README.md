**Lab Schedule Management System**  
A web-based scheduling system for managing university lab sessions, preventing instructor and lab room overlaps. Users can add schedules, check for conflicts, and export the schedule in **Excel and PDF** formats.  

### **Features**  
✅ Web-based interface for managing lab schedules  
✅ Conflict detection for instructors and lab rooms  
✅ Export schedule as **Excel (.xlsx)** and **PDF (.pdf)**  
✅ Built with **Flask (Python) & Vue.js**  

### **Tech Stack**  
- **Backend:** Flask, Pandas, SQLite/PostgreSQL  
- **Frontend:** Vue.js, HTML, CSS  
- **Export:** Pandas (Excel), ReportLab/WeasyPrint (PDF)  

### **Installation & Setup**  
1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/lab-schedule-system.git
   cd lab-schedule-system
   ```

2. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```

3. Run the Flask server:  
   ```bash
   python app.py
   ```

4. Open the frontend in a browser or Vue development server.
